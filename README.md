# SnackCenterProject
Snack Center Billing System in C and HTML Webpage for Fresh Bites<br>
This project demonstrates a simple **Snack Center application** built using **HTML** and **C language**.It is designed to show both **frontend presentation** and **backend billing logic**,suitable for beginners.<br>
##Project Overview<br>
The project is divided into **two parts**:<br>
###1.Snack Center Webpage(HTML)<br>
A Simple HTML webpage that displays a welcome heading for **Fresh Bites Snack Center**,an image representing the snack center<br>
This page acts as the **visual introduction** to the project.<br>
###2.Snack Center Billing System(C language)<br>
A console based billing system developed using C<br>
Features include:<br>
-Menu-driven snack selection.<br>
-Option to place multiple orders<br>
-Calculation and display of the final bill amount<br>
## How to View/Run the project<br>
### HTML Webpage<br>
1.Download the repository or clone it.<br>
2.open 'snackcenter.html' in any web browser.<br>
3.You will see the heading and image for fresh bites snack center.<br>
### C Billing Program<br>
1.Open 'snackcenter.c in **code::blocks** or any c compiler.<br>
2.Compile and run the project.<br>
3.Follow the on-screen menu to place orders and generate the bill..
